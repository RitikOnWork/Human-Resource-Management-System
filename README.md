# HRMS - Next-Generation Workforce Management System

A premium, state-of-the-art Human Resource Management SaaS platform designed to manage the modern enterprise workforce securely and efficiently. Transformed with a stunning, performance-driven dark aesthetic.

---

## 🚀 Key Features

*   **Intelligent Insights & Analytics**: Real-time mockups of workforce productivity directly on the glassmorphic dashboards.
*   **Geo-Fenced Time Tracking**: Simulated location-verified attendance mechanisms ensuring an audit-proof trail.
*   **AI-Powered Recruitment Pipeline**: Centralized hub mapping out the candidate lifecycle from resume parsing to the final interview.
*   **Multi-Role Application State**: 
    *   Unified, stateful Auth Flow (`/login` seamlessly dynamically merging with Signup flows).
    *   Granular role-based portals (Admin, HR Manager, Candidate, Employee).
*   **Lead Generation Engine**: Immersive "Book a Demo" funnel explicitly engineered for B2B conversions.
*   **Flawless Premium Aesthetic**: Pure custom CSS, using zero heavy libraries, ensuring maximum speed while boasting animated neon glow borders, 3D radial gradients, and an uninterrupted `#0b0f19` deep premium visual style.

---

## 🛠️ Technology Stack

This platform was built prioritizing clean code, lightweight execution, and strict custom UI/UX capabilities.
*   **Frontend**: React (18.x)
*   **Build Tool**: Vite
*   **Routing**: React Router DOM (v6)
*   **Styling**: 100% Custom CSS Modules utilizing CSS3 animations and CSS variables.
*   **Data Fetching**: Native Fetch API / Async-Await design patterns.

## 🏃 Getting Started (Local Development)

To run this platform locally on your machine:

1.  **Clone the Repository**
    ```sh
    git clone https://github.com/RitikOnWork/Human-Resource-Management-System.git
    cd Human-Resource-Management-System
    ```

2.  **Navigate directly to the Frontend Engine**
    ```sh
    cd frontend
    ```

3.  **Install Fast Dependencies**
    ```sh
    npm install
    ```

4.  **Launch the Development Server**
    ```sh
    npm run dev
    ```

    *The app will be instantly heavily compiled by Vite and accessible at [http://localhost:5173/](http://localhost:5173)*

## 📂 Design Architecture

The application UI is mapped around the concept of a seamless workspace.

*   `src/pages/Landing.jsx` - The main public facing engine demonstrating key features with high visual interaction.
*   `src/pages/auth/Login.jsx` - Our state-morphing authentication portal replacing standard redirects with immediate state flips.
*   `src/pages/admin/Dashboard.jsx` - Where the internal SaaS application actually starts displaying actual logic via the new `#111827` UI Cards.
*   `src/pages/landing.css` - The foundational spine of the public application where the `dark-premium` theme rules live.
